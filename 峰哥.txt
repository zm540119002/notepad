如果是实现了AutoCloseable的资源类，建议用jdk1.7后的try with resource写法，可以干掉finally的关闭方法：
例如：
try (BufferedReader br = new BufferedReader(new FileReader(fileName))) {
  //...
}
catch ( ... ) {
  //...
}
// finally{...}
